# Nostra Group V13

## Nouveauté
Dans Dashboard > Modification des pages, trois grands choix sont désormais affichés :

- Nostra Motors
- Nostra Circuit
- Jeux & Événements

Chaque choix ouvre uniquement les pages de sa partie du site. Les pages Nostra Motors et Jeux & Événements sont maintenant réellement modifiables depuis le dashboard, comme les pages Nostra Circuit.

Aucun nouveau code SQL n'est nécessaire si l'éditeur de pages est déjà activé dans Supabase.
