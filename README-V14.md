# Nostra Group V14

## Correction

Le Dashboard permet maintenant de créer des pages personnalisées dans les trois parties :

- Nostra Motors
- Nostra Circuit
- Jeux & Événements

Dans **Dashboard → Modification des pages**, ouvre la partie voulue puis descends jusqu'à **Pages personnalisées**.

Une page créée apparaît automatiquement dans le menu de la partie concernée. Elle peut ensuite être modifiée, masquée ou supprimée depuis le même écran.

Aucun nouveau script SQL n'est requis.
